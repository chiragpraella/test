## Getting Started

1. Clone the repository
2. Install Shopify CLI: `npm install -g @shopify/cli @shopify/theme`
3. Start development server using `shopify theme dev`

## Development Guidelines

- Follow the established code structure
- Use the provided component patterns
- Maintain consistent naming conventions
- Follow the documentation format for new features
- Use Shopify CLI for theme development
- Run theme check before committing changes
